Window Object

window object is the top most object in the DOM hierarchy. It represents the browsers frame that displays the contents of a webpage. Whenever a window appears on the screen to display the contents of the document, the window object is created.

Syntax:- window.property-name;

Properties:-

  • console
  • frames[]
  • History
  • Length
  • Location
  • Name
  • Window


Document Object

Document Object represents a webpage that is loaded in the browser. By accessing the document object, we can access the elements in the HTML page. With the help of document objects, we can add dynamic content to our webpage

Syntax:- document.property-name;

Properties:-

  • links
  • images
  • scripts
  • body
  • anchor
  • activeElement
  • cookie
